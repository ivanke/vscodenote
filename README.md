## 設定 Settings
### 分類
vscode 的 Settings 依作用範圍的不同，分為兩類
* User
    * 作用範圍：全域
* Workspace
    * 作用範圍：工作目錄
    * vscode 會將設定存在該工作目錄的 **.vscode/settings.json** 檔案中
### 如何進入設定
1. 按 **Ctrl + Shift + P**，輸入關鍵字：**settings**
2. 選擇 **Preferences: Open User Settings**，即可進入 **Settings** 畫面
